## 🚀 最初にやってほしいこと

1. **プロジェクト初期化**
   ```bash
   mkdir my-game && cd my-game
   npx degit kumakun8325/sdd-templates .
   ```

2. **SDDドキュメントのプレースホルダーを置換**
   - `{{PROJECT_NAME}}` → 実際のプロジェクト名
   - `{{DESCRIPTION}}` → プロジェクト説明
   - その他

3. **開発環境構築**
   ```bash
   npm init -y
   npm install phaser typescript webpack webpack-cli ts-loader
   ```

4. **最初のシーン作成**
   - BootScene → PreloadScene → TitleScene
